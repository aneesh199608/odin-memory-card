# Memory Card Game

A classic memory challenge game built with **React + Vite**, styled using [NES.css](https://nostalgic-css.github.io/NES.css/), and powered by the [PokeAPI](https://pokeapi.co/). Try to click each Pokémon **only once** — cards shuffle after every click!

[screenshot](./src/assets/memory-card-screenshot.png) 

## Technologies Used

- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [NES.css](https://nostalgic-css.github.io/NES.css/)
- [PokeAPI](https://pokeapi.co/)
